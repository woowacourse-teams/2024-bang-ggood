## 🏡 서비스 소개

방을 처음 구하는 자취 초보라면 이런 고민, 해본 적 있지 않나요?

> 🕒 “방 보는 시간은 단 10분… 어디서부터 봐야 하지?”  
> 😵‍💫 “내 눈에는 다 좋아 보이는데…”  
> 🤔 “방금 본 집들이 다 비슷해서 기억이 안 나네..”

방끗은 이 고민들을 해결하기 위해 만들어졌습니다.  
방을 둘러볼 때 놓치기 쉬운 포인트들을 **체크리스트**로 제공하고,  
당신만의 맞춤형 리스트를 만들어 **최적의 방을 선택**할 수 있도록 도와줘요.
